# ML-group-project

CreditSense final project workspace.

## Layout

- `data/creditsense-ai1215/`: raw train/test/sample CSVs
- `task_a/`: Task A support notebooks, scripts, reports, and artifacts
- `task_b/`: Task B notebook and regression artifacts
- `submissions/`: combined competition-style submission files
- `docs/`: reference documents
- `taskA_nn.ipynb`: current Task A experiment notebook at the repo root
- `TaskA.md`: Task A running model log
